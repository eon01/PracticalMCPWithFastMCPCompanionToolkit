# FastMCP in Depth: Building "The Puppy Guide" MCP


## Preparing the Development Environment


```bash
# Create a new directory for our project
mkdir -p $HOME/workspace/puppy_guide/server

# Navigate into it
cd $HOME/workspace/puppy_guide/server

# Initialize a new UV project with Python 3.12
uv init --bare --python=3.12

# Install the FastMCP library
uv add "fastmcp==3.4.2"
```


```json
cat > fastmcp.json << 'EOF'
{
  "$schema": "https://gofastmcp.com/public/schemas/fastmcp.json/v1.json",
  "source": {
    "path": "main.py",
    "entrypoint": "mcp"
  },
  "environment": {
    "type": "uv",
    "python": ">=3.12"
  },
  "deployment": {
    "transport": "http",
    "host": "localhost",
    "port": 8000,
    "log_level": "INFO"
  }
}
EOF
```


## The FastMCP Class


```python
from fastmcp import FastMCP

mcp = FastMCP(
    name="Puppy Guide",
    list_page_size=10,
    instructions="You are a helpful assistant that provides information "
                  "about the real age of dogs in human years."
)
```


```bash
uv add "python-dotenv==1.2.2"
```


```bash
cat > $HOME/workspace/puppy_guide/server/.env << EOF
MCP_NAME="Puppy Guide"
MCP_LIST_PAGE_SIZE=10
MCP_INSTRUCTIONS="You are a helpful assistant that provides information about the real age of dogs in human years."
EOF
```


```python
cat > $HOME/workspace/puppy_guide/server/main.py << EOF
import os
from dotenv import load_dotenv
from fastmcp import FastMCP

load_dotenv()

mcp = FastMCP(
    name=os.getenv("MCP_NAME"),
    list_page_size=int(os.getenv("MCP_LIST_PAGE_SIZE", "10")),
    instructions=os.getenv("MCP_INSTRUCTIONS")
)
EOF
```
