## UV: The Fast Python Packaging Tool


```bash
# Install uv 0.10.2
curl -LsSf https://astral.sh/uv/0.10.2/install.sh | sh

# Reload the shell configuration
source ~/.bashrc

# Verify the installation
uv --version
```


```bash
# Create a project directory
mkdir $HOME/testing-uv

# Change to the project directory
cd $HOME/testing-uv

# Initialize a new uv project
uv init

# Install a package (for example, requests)
uv add requests

# Check the installed packages.
# requests and its dependencies should be listed
uv pip list
```


```bash
# Remove the requests package
uv remove requests

# List the installed packages again to confirm that requests has been removed
uv pip list
```


```bash
# Change directory
cd $HOME/testing-uv

# Remove the virtual environment
rm -rf .venv

# Change directory
cd $HOME

# Remove the project directory
rm -rf testing-uv
```

## NPM and Node.js


```bash
# Download and install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# in lieu of restarting the shell
\. "$HOME/.nvm/nvm.sh"

# Download and install Node.js
nvm install 24

# Install npx globally
npm install -g npx 2>/dev/null || echo "npx is already installed"
```
